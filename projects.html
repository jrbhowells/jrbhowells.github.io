Coffee Table

Part of my A-level course, I made this coffee table in walnut and ash, with a tempered glass top. The joinery was by far the most complex task: the bottom of each leg features a hybrid mitre/bridle joint, and the tabletop itself is joined using a mitre joint with ash keys. All joinery had to be cut by hand due to tool-use restrictions.

Secret Compartment Box

As a free-time project in my school's DT workshop, I made a small box in sapele, beech and ash with my initials inlayed in the top. More details coming soon.

Recycled Teak Garden Bench

My iGCSE Resistant Materials (DT) coursework, this bench was made from fully recycled teak: the boards had been used for flooring. They were quite thin so needed laminating together into beams. More detail coming soon.

This Website

This website is my own creation. I built it using HTML and CSS, with some JavaScript for interactive elements, and it is made available through GitHub Pages. Because I've coded it from scratch, with no frameworks, there may be some compatibility issues with some browsers etc. - if you find any, please let me know! I have some experience with JavaScript and PHP, as well as (very limited) knowledge of Flask and Django, although as there is no backend on this site I haven't used any of these.

Machine Learning Weather Forecast

A work-in-progress project, I am working on predicting the weather using machine learning, and displaying this on a website. Data come from the Met Office's free DataPoint API, which provides hour-by-hour readings for current weather at around 150 locations nationwide. More detail coming soon.

Calendar / Reminders Webapp

A complete HTML, CSS and JavaScript webapp for organising tasks in a calendar format. This project was part of a computing module I took as part of my degree, and used an neDB database to store information. Meshing a computer's idealised vision of time with a calendar (e.g. accounting for leap years) proved to be the most challenging part.

Coronavirus Data Scraper

In early February 2020, 2019-nCoV (now called COVID-19) was spreading rapidly. Accurate data were hard to find at the time, so I decided to create an extremely simple scraping algorithm in Python to log current COVID cases, hospitalisations and deaths.

The program has been running since, saving data to an Excel spreadsheet. Far more accurate data are now available, but I have maintained the program and kept it running for interest. The above graph shows cases, hospitalisatons and deaths over time, and the CFR (case fatality rate, reported deaths per reported case) is below. The large gap between days ~640 and ~860 are where I lost data to a faulty SSD.

Graphs last updated 03/02/2022.

Fully Sustainable Eco-Home

Part of my Pre-U in Art & Design (DT), this was a project I'd wanted to do for a long time. I analysed the exact needs of a 2-bedroom home and devised ways of providing all of this in a fully-sustainable way, with no reliance on external utilities. In addition to this, I designed and rendered the home and produced a specification for its contsruction. More detail coming soon.

CareWare: Space Saving in Small Kitchens

A design project forming part of my degree, I worked in a group to create a human-centred, space-saving device for small kitchens. The final product was a multipurpose sink cover, which allowed the sink to be used as extra counter space or a chopping board - or could be rotated through 140 degrees to become a drying rack beside the sink. More detail coming soon.

Christmas in a Box PSS

Part of a sustainable design module for my degree, I worked in a group to produce a product-service system focused on delivering a more sustainable Chirstmas at a lower price than existing options. More detail coming soon.

RevEng: Reverse Engineering

A deliverable for a Production and Materials module, I tore down a cheap trim router, analysing its components, material make-up and manufacturing processes, before making recommendations for its improvement. More detail coming soon.

Carbon-neutral Housing Study

Part of a thermofluids module, I analysed a typical rural bungalow's energy needs, and came up with several interventions to improve its energy performance. These ranged from better insulation to harnessing hydroelectricity and heat pumps. More detail coming soon.

Vehicle Aerodynamics Simulation

A study involving designing a car in SolidWorks and analysing its aerodynamic performance. I sketched my concept car before importing these into SolidWorks and using projected sketches to create a 3D object. This could then be analysed using CFD techniques. More detail coming soon.
